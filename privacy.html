Política de Privacidade — CASA

Última atualização: 2026-06-15

A CASA é uma aplicação pessoal de gestão financeira doméstica, de utilização privada pelo seu titular. Esta página descreve como os dados são tratados.

1. Que dados são tratados

  • Movimentos financeiros, categorias e saldos introduzidos pelo utilizador.
  • Quando o utilizador opta por ligar uma conta bancária, são obtidos apenas para leitura (AIS) os dados de conta e transações, com o seu consentimento explícito.

2. Acesso bancário (open banking)

O acesso às contas é feito através da Enable Banking, prestador de serviços de informação sobre contas (AISP) licenciado, ao abrigo da diretiva PSD2. A autenticação é efetuada diretamente no banco do utilizador (autenticação forte/SCA); a aplicação nunca tem acesso às credenciais bancárias. O acesso é exclusivamente de leitura — a aplicação não inicia pagamentos nem movimenta fundos. O consentimento é temporário (tipicamente 90 dias) e renovável pelo utilizador.

3. Onde ficam os dados

  • Os dados financeiros são guardados localmente no dispositivo do utilizador (armazenamento do navegador) e, opcionalmente, sincronizados na conta Dropbox pessoal do utilizador.
  • Uma função intermediária (Supabase Edge Function) serve apenas para comunicar com a API do banco; não armazena dados financeiros nem transações.
  • Os dados não são partilhados com terceiros nem usados para publicidade ou perfilagem.

4. Direitos do utilizador

O utilizador pode, a qualquer momento, desligar o acesso bancário, apagar os dados locais e revogar o consentimento junto do banco ou da Enable Banking.

5. Contacto

Para questões de privacidade, contacte o titular da aplicação.
